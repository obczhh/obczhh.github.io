AdaptixC2编译到运行

AdaptixC2

Adaptix 是一个为渗透测试人员设计的可扩展的后渗透和对抗模拟框架。Adaptix 服务器使用 Golang 编写，以允许操作员的灵活性。GUI 客户端使用 C++ QT 编写，使其可以在 Linux、Windows 和 MacOS 操作系统上使用。

Features 功能特性

  • Server/Client Architecture for Multiplayer Support
    支持多人游戏的服务器/客户端架构
  • Cross-platform GUI client
    跨平台 GUI 客户端
  • Fully encrypted communications
    完全加密的通信
  • Listener and Agents as Plugin (Extender)
    监听器和代理作为插件（扩展器）
  • Client extensibility for adding new tools
    客户端可扩展性，用于添加新工具
  • Task and Jobs storage 任务和作业存储
  • Files and Process browsers
    文件和进程浏览器
  • Credentials, Targets and Screenshots Managers
    凭证、目标和屏幕截图管理器
  • Socks4 / Socks5 / Socks5 Auth support
    Socks4 / Socks5 / Socks5 认证支持
  • Local and Reverse port forwarding support
    本地和反向端口转发支持
  • BOF support BOF 支持
  • Linking Agents and Sessions Graph
    连接代理和会话图
  • Agents Health Checker 代理健康检查器
  • Agents KillDate and WorkingTime control
    代理杀毒日期和工作时间控制
  • Windows/Linux/MacOs agents support
    Windows/Linux/MacOs 代理支持
  • Remote Terminal 远程终端
  • AxScript Engine AxScript 引擎

扩展套件

Official Extension-Kit on GitHub

当前扩展者

  • HTTP/S Beacon Listener HTTP/S 信标监听器
  • SMB Beacon Listener SMB 信标监听器
  • TCP Beacon Listener TCP 信标监听器
  • Beacon Agent 信标代理
  • TCP/mTLS Gopher Listener TCP/mTLS Gopher 监听器
  • Gopher Agent

预安装：

以上安装完毕后即可make (go env -w GOPROXY=https://mirrors.aliyun.com/goproxy/,direct)

AdaptixServer 和 Extenders (goplugins) 必须使用相同版本的 golang 包进行构建。

所有编译后的文件都将位于 dist 目录中。

Client 客户端

Linux / MacOS

AdaptixClient 是从 cmake 配置文件编译的。

AdaptixClient 文件将位于 dist 目录中。

Windows

从官方网站下载 QT 在线安装程序：https://www.qt.io/download-qt-installer-oss。

(这里安装需要一直手动重试 然后才能安装成功 )

然后将AdaptixClient复制出来 QT打开CMakeLists.txt 文件 打开后config之后提示更新在更新就行了

项目中添加release模式

为客户端创建一个目录，所有依赖项将复制到该目录。将 AdapticClient.exe 可执行文件移动到此目录，打开 cmd.exe 并运行 windeployqt.exe。即可直接运行！
