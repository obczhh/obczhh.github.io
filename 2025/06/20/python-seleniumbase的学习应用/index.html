python-seleniumbase的学习应用

安装，这里使用Ubuntu，windows物理机跟谷歌浏览器有冲突，创建目录创建venv虚拟环境后github或者pip安装后，开始应用。

中间的参数可以根据需求调节，具体如下

参数名 类型 默认值 说明
browser str "chrome" 浏览器类型："chrome"、"edge"、"firefox"、"safari" 等
uc bool False 是否使用 undetected-chromedriver 防止检测
proxy str None 设置 HTTP 或 SOCKS 代理地址，如："http://127.0.0.1:7890"
headless bool False 是否无头运行（不显示浏览器界面）
incognito bool False 是否以隐身模式启动浏览器
maximize bool True 启动后是否最大化浏览器窗口
locale_code str None 浏览器语言设置（如 "zh-CN"、"en-US"）
agent str None 自定义 User-Agent
cap_file str None 加载自定义浏览器 capabilities 的 json 文件
extension_zip str None 加载 Chrome 插件 .zip 文件
extension_dir str None 加载 Chrome 插件的目录
start_page str None 浏览器启动时加载的初始页面 URL
disable_csp bool False 是否禁用 CSP（内容安全策略）
enable_sync bool False 启用 Google 登录同步功能（需要配置好用户数据）
use_auto_ext bool True 使用 seleniumbase 自带的自动化扩展
no_sandbox bool False 禁用 sandbox 模式（有时 root 用户需要）
disable_gpu bool False 禁用 GPU 加速（用于某些系统兼容）

语法 匹配 HTML 标签

浏览器操作相关

方法名 说明 示例代码
open(url) 打开指定网址 sb.open("https://www.baidu.com")
refresh() 刷新当前页面 sb.refresh()
go_back() 浏览器后退 sb.go_back()
go_forward() 浏览器前进 sb.go_forward()
get_current_url() 获取当前页面URL url = sb.get_current_url()
get_page_title() 获取当前页面标题 title = sb.get_page_title()
save_screenshot(filename) 保存当前页面截图 sb.save_screenshot("screenshot.png")
quit() 关闭浏览器并结束所有会话 sb.quit()

标签页 / 窗口管理

方法 / 属性 说明 示例代码
driver.current_window_handle 获取当前窗口句柄 handle = sb.driver.current_window_handle
driver.window_handles 获取所有打开窗口的句柄列表 handles = sb.driver.window_handles
driver.switch_to.window(handle) 切换到指定窗口句柄 sb.driver.switch_to.window(handles[0])
driver.close() 关闭当前窗口 sb.driver.close()

元素定位与操作

方法 说明 示例代码
click(selector) 点击页面元素，selector为CSS选择器 sb.click("#submit-btn")
type(selector, text) 输入文字到元素 sb.type("#search-box", "seleniumbase\n")
get_text(selector=None) 获取元素文本，未传selector则获取整个页面文本 text = sb.get_text("#main")
assert_element(selector) 断言页面存在指定元素 sb.assert_element("#login-form")
wait_for_element(selector, timeout=10) 等待元素出现，超时抛异常 sb.wait_for_element(".result-item", timeout=5)
is_element_visible(selector) 判断元素是否可见 visible = sb.is_element_visible("#menu")
hover(selector) 鼠标悬停元素 sb.hover(".dropdown")

其他实用方法

方法 说明 示例代码
execute_script(js_code) 执行JavaScript代码 sb.execute_script("window.scrollTo(0, document.body.scrollHeight);")
get_attribute(selector, attribute) 获取元素指定属性的值 value = sb.get_attribute("#input", "value")
select_option_by_text(selector, text) 通过文本选择下拉框选项 sb.select_option_by_text("#dropdown", "Option 2")
wait_for_ready_state_complete(timeout=30) 等待页面加载完成 sb.wait_for_ready_state_complete()

额外说明

  • 选择器（selector）可以使用CSS选择器，如：#id、.class、[attr="value"]。
  • get_text() 无参数时获取整个页面的可见文本，传入参数时获取指定元素的文本。
  • 大部分操作会自动等待元素出现（implicit wait），但你也可以用wait_for_element控制等待时长。
  • SeleniumBase底层封装了selenium.webdriver，可以直接通过sb.driver访问selenium的原生方法和属性。
